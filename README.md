# ck_programmkalender

This project is intended to distribute the CampusKino HS Harz Programmkalender for different calendar apps. 

It is deployed as a website displaying 4 ways to add the programmkalender (specified for each app).
- For Google Calendar users (Link)
- For Apple Calendar users (QR Code + instructions)
- As downloadable .ics (Link)
- For anyone as iCal/ICS-Stream. (URL)
